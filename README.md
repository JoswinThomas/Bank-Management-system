# Bank Management System in C++

A console-based Bank Management System developed using C++.

## Features
- Create Account
- Deposit Money
- Withdraw Money
- Transfer Funds
- Transaction History
- Account Details Display
- Close Account

## Technologies Used
- C++
- Object Oriented Programming
- STL (vector, string, etc.)

## How to Run

Compile:
```bash
g++ bank.cpp -o bank
